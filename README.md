# RenPyUtil
> 一个Ren'Py工具包，提供了一系列基于Ren'Py的功能类，以供Ren'Py开发者调用。

**:sparkles: 欢迎Issue, Fork和PR**

## :cd: 如何使用
把[RenPyUtil](./RenPyUtil)文件夹置于你的游戏（game）目录下即可在脚本中调用。

## :rocket: 功能概览
- [x] 内置文档，符合PEP257，更易上手。
- [x] 高级角色类，拥有更多可能。。
- [x] 基于socket的TCP协议的多线程网络通信支持。
- [ ] 网络内容更新支持。
- [ ] 数据管理器，使管理游戏数据更加方便。
